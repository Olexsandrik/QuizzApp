# QuizApp

A full-stack quiz application for creating and taking quizzes.


## Tech Stack
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS/ frontend folder
- **Backend:** Express.js, Prisma ORM/ backend folder
- **Database:** PostgreSQL

## Features
- Create and delete quizzes
- Add questions with multiple types (boolean, input, checkbox)
- Take quizzes interactively
